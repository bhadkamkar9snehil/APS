## Workbook Tools

This folder contains workbook maintenance utilities that are not part of the normal APS runtime path.

- `setup_excel.py` rebuilds or repairs workbook helper sheets and workbook-side setup.
- `create_algorithm_config_sheet.py` recreates the `Algorithm_Config` sheet with default parameters.
- `master_data_fixer.py` applies cleanup and routing fixes to workbook master data.

These scripts are intended for workbook maintenance tasks, not for day-to-day planning runs.
