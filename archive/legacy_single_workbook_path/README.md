# Archived Legacy Path

This folder contains the retired `APS_Steel_Template` and `APS_Steel_Template_v2` workbook flow.

Archived here:

- Old workbooks: `APS_Steel_Template*.xlsx` and `APS_Steel_Template*.xlsm`
- Old root-level scripts: `loader.py`, `bom_explosion.py`, `capacity.py`, `scheduler.py`, `scenario_runner.py`, `simpy_engine.py`

These files are preserved for reference only.
They are not part of the supported APS runtime.

Supported runtime:

- Workbook: `APS_BF_SMS_RM.xlsm`
- Builder: `python build_template.py`
- Setup: `python setup_excel.py`
- Run: `python run_all.py all`
